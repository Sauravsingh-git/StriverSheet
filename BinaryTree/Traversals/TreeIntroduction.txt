compter binary tree : It is a type of tree where all the levels of the tree are filled
with nodes ecept the alst level. The last level must have each node as left as possible
perfect binary tree : It is a type of tree in which all the leaft nodes should be on the

balanced binary tree : In a balanced binary tree, the height of the tree should be at 
max(logN), where N is the number of nodes. The tree should not be skewed.
degenerate tree : A degenerate tree is a skewed tree that is similar to a linkedlist where
each node is connected to left of the previous node and the tree goes leftwares only
